#### install

```bash
git clone https://github.com/plantain-00/news-fetcher-release.git . --depth=1 && npm i --production
```

Set environment variable NEWS_FETCHER_KEY

```bash
node dist/start.js
```

Then open `http://localhost:9994` in your browser.
